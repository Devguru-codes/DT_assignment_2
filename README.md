# Corporate Leave Approval System

Backend prototype for an IT Prototyping of Approval & Authorization System using a corporate leave workflow.

## Phase 1 Scope

- role models for employee, HR manager, and admin
- secure login with password hashing
- session-based authentication
- role-based dashboard routing

## Setup

```powershell
venv\Scripts\python -m pip install -r requirements.txt
venv\Scripts\python app.py
```

## Demo Accounts

- Employee: `employee@corp.local` / `Employee@123`
- HR Manager: `manager@corp.local` / `Manager@123`
- Admin: `admin@corp.local` / `Admin@123`
